# Add contestのところについてまとめる

# External ID 
基本無視してよさそう。（何もしなかったら勝手に生成される）

# Shortname/Name
ショートネームは画面右上に表示されるやつ（最初だったらdemo）。Nameはリーダーボードとかに表示されるから正式な奴がいいか。

# ctivate time / Deactivate time
参加者がコンテストのサイトにアクセスできる時間のこと。
Deactivate timeはその時間以降参加者は順位表とかの確認ができない。まあ3時間とかにしたらいいのかな？

# Start time / End Time
コンテストのスタート時間と終了時間

# Enable public scoreboard
参加者以外も順位表が見れるかどうか。あってもよさそう。
